    TUrgency - Message urgency
    TStatus - Status
    TBuySell - Directivity of the order

    TSystemID - Trade System identifier
    TTimeStatus - Trading time event status
    TBoolInt - Boolean enumeration type

    Таблицы

Перечислимые типы

    TPeriod - Trading period (длина: 1, тип: ekGroup)

      List of trading periods.

      Константа Значение Описание
      O   Open period
      N   Normal trading period
      C   Closing period

    TUrgency - Message urgency (длина: 1, тип: ekCheck)

      List of available types of the messages.

      Константа Значение Описание
          Nornal message
      U   Urgent message

    TStatus - Status (длина: 1, тип: ekCombo)

      List of possible values of the status of fulfilment of trade operations.

      Константа Значение Описание
      A   Active (operations are permitted)
      S   Suspended (operations are prohibited)

    TBuySell - Directivity of the order (длина: 1, тип: ekGroup)

      List of acceptable values determining a directivity of the order.

      Константа Значение Описание
      B   Buy
      S   Sell

    TTradingStatus - Trading status of the security (длина: 1, тип: ekGroup)

      List of trading statuses of the security.

      Константа Значение Описание
      N   No trading
      O   Open period
      C   Trading session is closed
      F   Closing period
      B   Break of a tradind session
      T   Normal trade session

    TYesNo - Boolean type (длина: 1, тип: ekCheck)

      Константа Значение Описание
      N   No
      Y   Yes

    TQuoteBasis - Security price quote type (длина: 1, тип: ekCombo)

      Константа Значение Описание
      R   The price is specified for unit of the security
      A   The price is specified for a face value of the security (currency market)
      F   The price is specified in percentage of a face value

    TSecType - Типы ценных бумаг (длина: 1, тип: ekCombo)

      Константа Значение Описание
      1   Акции обыкновенные ( )
      2   Акции привилегированные
      3   Ценная бумага РФ
      4   Ценные бумаги муниципальные и субъектов РФ
      5   Ценные бумаги ЦБ РФ
      6   Корпоративные облигации
      7   Облигации МФО
      8   Биржевые облигации
      9   Паи открытых ПИФов
      A   Паи интервальных ПИФов
      B   Паи закрытых ПИФов

    TSystemID - Trade System identifier (длина: 1, тип: ekCombo)

      Константа Значение Описание
      C   Currency market Trading System
      P   Securities market Trading System

    TTimeStatus - Trading time event status (длина: 1, тип: ekCombo)

      List of statuses of trading session events.

      Константа Значение Описание
      A   Actual
      L   Executed (lapsed)

    TBoolInt - Boolean enumeration type (длина: 1, тип: ekCheck)

      Константа Значение Описание
      0   No
      1   Yes

Интерфейс "Info" - "Information interface"

Таблицы

    ALL_TRADES - All trades (обновляемая)

      Таблица содержит общую информацию (без указания сторон по сделке) по всем сделкам купли/продажи финансовых инструментов, совершенным в Торговой Системе с начала торговой сессии / The table contains the public information (without the indicating of the parties on the trade) under all trades of sale and purchase of securities accomplished in a MICEX Trading System from a beginning of trade session.

      Входных полей нет
      Выходное поле Описание Тип Размер Примечание
      TRADENO Trade number INTEGER 12 Идентификационный номер сделки в Торговой Системе / The system generated trade number
      TRADETIME Time TIME 6 Время регистрации сделки в Торговой Системе / The time that the trade was matched.
      SECBOARD Board CHAR 4 Идентификатор режима торгов для финансового инструмента / The board code for this trade.
      SECCODE* Security CHAR 12 Идентификатор финансового инструмента / The security code for this trade.
      PRICE Price PRICE 9 Цена за одну ценную бумагу / The trade price.
      QUANTITY Quantity INTEGER 10 Объём сделки, выраженный в лотах / The trade quantity (in lots).
      VALUE Value PRICE 16 Объём сделки, выраженный в руб / The trade volume (in currency units).
      ACCRUEDINT Accrued interest PRICE 16 Процентный доход, выраженный в руб. / The accrued interest (in currency units).
      YIELD Yield NUMERIC 9.2 Доходность, рассчитанная по цене сделки / The yield of the security.
      PERIOD Trading period TPeriod 1 Период торговой сессии, когда была заключена сделка / Period of trading session, when the trade took place.
      SETTLECODE Settlement code CHAR 12 Код расчётов по сделке /The code of settlement for the trade.

    BCMESSAGES - Broadcast messages (обновляемая)

      Таблица содержит информацию о сообщениях, полученных от других участников торгов с момента начала торговой сессии / The table contains the messages received from other users from the beginning of trading session.

      Входных полей нет
      Выходное поле Описание Тип Размер Примечание
      URGENCY Urgency TUrgency 1 Срочность или важность сообщения / Determines whether the message is urgent or not.
      FROMUSER User CHAR 12 Идентификатор пользователя, отправившего сообщение / The identifier of the user who sent the message.
      MSGTIME Time CHAR 8 Время, когда было отправлено сообщение / The time the user originally sent the message.
      MSGTEXT Text CHAR 256 Текст сообщения / The text of the message.

    BOARDS - Boards (не обновляемая)

      Таблица содержит информацию о режимах торгов. Под режимом понимается набор правил проведения торгов, определяющих допустимые типы заявок, регламент торгов, виды расчетов и пр. Каждый режим представлен в таблице одной строкой / The table contains the information on boards. Board is understood as a set of trading rules determining allowed types of the orders, codes of settlements and etc. Each board is shown in the table by one row.

      Входных полей нет
      Выходное поле Описание Тип Размер Примечание
      BOARDID Board identifier CHAR 4 Идентификатор режима торгов / The identifier of the board.
      BOARDNAME Name CHAR 30 Наименование режима / Board name.
      STATUS Status TStatus 1 Индикатор "торговые операции разрешены/запрещены" / The indicator "the trade operations permit / prohibit".
      MARKETID Market CHAR 4 Идентификатор рынка, к которому относится данный режим / The identifier of the market for the given board.
      LATNAME English name CHAR 30 Наименование режима торгов на английском языке / Board name in English.

    ORDERBOOK - Order book (обновляемая, очищать при обновлении)

      Таблица содержит информацию о заявках с лучшими ценами (отдельно на покупку и на продажу) по данному финансовому инструменту.       Входное поле Описание Тип Размер Примечание

      Выходное поле Описание Тип Размер Примечание

      SECCODE* Security CHAR 12 Идентификатор финансового инструмента / The identifier of the security.
